# Инструкция: Страница заполнения договора

## Цель
Создать **один автономный HTML-файл** (`contract-filler.html`) — страницу для заполнения договора через красивый интерфейс в стиле CRM (`index.html`). Файл должен работать на GitHub Pages без бэкенда.

## 2. Дизайн-система (КОПИРОВАТЬ из index.html)

Скопировать **полностью** блок CSS-переменных `:root{...}` и `[data-theme=dark]{...}` из `index.html` (строки 10-35). Это:
- Цвета: `--bg`, `--surface`, `--blue`, `--green`, `--text` и т.д.
- Шрифты: `--font:'Golos Text'`, `--mono:'JetBrains Mono'`
- Радиусы: `--r:10px`, `--rl:14px`

```html
<link href="https://fonts.googleapis.com/css2?family=Golos+Text:wght@400;500;600&family=JetBrains+Mono:wght@400;500&display=swap" rel="stylesheet">
```

Интерфейс должен быть **миниатюрным** — компактная карточка по центру экрана, максимальная ширина 520px.

## 6. Функция «Число прописью» (РЕАЛИЗАЦИЯ)

Написать функцию `numberToWords(n)` на русском языке:
- Принимает целое число, возвращает строку прописью
- Пример: `150000` → `"сто пятьдесят тысяч"`
- Поддержка от 0 до 999 999 999
- Учитывать склонение: "одна тысяча", "две тысячи", "пять тысяч"

Массивы для реализации:
```javascript
const ones = ['','один','два','три','четыре','пять','шесть','семь','восемь','девять'];
const onesF = ['','одна','две','три','четыре','пять','шесть','семь','восемь','девять'];
const teens = ['десять','одиннадцать','двенадцать','тринадцать','четырнадцать',
               'пятнадцать','шестнадцать','семнадцать','восемнадцать','девятнадцать'];
const tens = ['','','двадцать','тридцать','сорок','пятьдесят',
              'шестьдесят','семьдесят','восемьдесят','девяносто'];
const hundreds = ['','сто','двести','триста','четыреста','пятьсот',
                  'шестьсот','семьсот','восемьсот','девятьсот'];
```

Склонение "тысяч": 1→"тысяча", 2-4→"тысячи", 5-20→"тысяч". Для тысяч — женский род (одна, две).

Склонение "миллионов": 1→"миллион", 2-4→"миллиона", 5-20→"миллионов".

## 7. Функция формирования инициалов

```javascript
function getInitials(fullName) {
    const parts = fullName.trim().split(/\s+/);
    if (parts.length < 2) return fullName;
    const surname = parts[0];
    const initials = parts.slice(1).map(p => p[0].toUpperCase() + '.').join('');
    return initials + ' ' + surname;
}
```

## 8. Функция формирования даты

```javascript
function formatDate() {
    const months = ['января','февраля','марта','апреля','мая','июня',
                    'июля','августа','сентября','октября','ноября','декабря'];
    const d = new Date();
    const day = String(d.getDate()).padStart(2,'0');
    return '«' + day + '» ' + months[d.getMonth()] + ' ' + d.getFullYear() + ' г.';
}
```

## 9. Авто-обновление полей (event listeners)

```javascript
// При вводе ФИО → обновить инициалы
document.getElementById('client_name').addEventListener('input', function() {
    document.getElementById('initials').value = getInitials(this.value);
});

// При вводе суммы → обновить сумму прописью
document.getElementById('amount').addEventListener('input', function() {
    const n = parseInt(this.value) || 0;
    document.getElementById('amount_words').value = numberToWords(n) + ' рублей';

// При вводе предоплаты → обновить предоплату прописью
document.getElementById('prepayment').addEventListener('input', function() {
    const n = parseInt(this.value) || 0;
    document.getElementById('prepayment_words').value = numberToWords(n) + ' рублей';
});

// Дата — заполнить при загрузке страницы
document.getElementById('contract_date').value = formatDate();
```

## 11. Возможность загрузки своего шаблона

Добавить секцию «Шаблон договора»:
1. Текст: "По умолчанию — встроенный договор Mebel Grand СПБ"
2. `<input type="file" accept=".docx">` — "Загрузить свой шаблон"
3. При загрузке — `FileReader.readAsArrayBuffer`, сохранить в `customTemplate`
4. Показать имя файла и кнопку "Сбросить" (вернуть встроенный)

```javascript
let customTemplate = null;

fileInput.addEventListener('change', function(e) {
    const file = e.target.files[0];
    if (!file) return;
    const reader = new FileReader();
    reader.onload = function(ev) {
        customTemplate = ev.target.result;
        // Показать имя файла, показать кнопку "Сбросить"
    };
    reader.readAsArrayBuffer(file);
});
```

## 12. Генерация и скачивание договора

```javascript
function generateContract() {
    try {
        let zipData;
        if (customTemplate) {
            zipData = new PizZip(customTemplate);
        } else {
            zipData = new PizZip(atob(DEFAULT_TEMPLATE_B64));
        }

        const doc = new window.docxtemplater(zipData, {
            paragraphLoop: true,
            linebreaks: true
        });

        const data = {
            'Номер_договора': document.getElementById('contract_number').value,
            'Дата_договора': document.getElementById('contract_date').value,
            'ФИО_клиента': document.getElementById('client_name').value,
            'Сумма_цифрами': document.getElementById('amount').value,
            'Сумма_прописью': document.getElementById('amount_words').value,
            'Предоплата_цифрами': document.getElementById('prepayment').value,
            'Предоплата_прописью': document.getElementById('prepayment_words').value,
            'Паспорт': document.getElementById('passport').value,
            'Адрес_регистрации': document.getElementById('registration').value,
            'Адрес_доставки': document.getElementById('delivery').value,
            'Телефон': document.getElementById('phone').value,
            'Email': document.getElementById('email').value,
            'Инициалы_Фамилия': document.getElementById('initials').value
        };

        doc.render(data);

        const out = doc.getZip().generate({
            type: 'blob',
            mimeType: 'application/vnd.openxmlformats-officedocument.wordprocessingml.document'
        });

        const a = document.createElement('a');
        a.href = URL.createObjectURL(out);
        a.download = 'Договор_' + (document.getElementById('client_name').value || 'клиент') + '.docx';
        a.click();
        URL.revokeObjectURL(a.href);

        toast('Договор сгенерирован!', 'success');
    } catch(e) {
        toast('Ошибка: ' + e.message, 'error');
    }
}
```

> [!IMPORTANT]
> Плейсхолдеры в `data` — БЕЗ фигурных скобок `{{}}`. docxtemplater сам ищет `{{Ключ}}` в документе и подставляет значение по ключу `Ключ`.
